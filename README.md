# Dano Airlines Customer Satisfaction Analysis

## Project Overview
Dano Airlines experienced a decline in customer satisfaction below 50%.  
This project analyzes passenger survey and operational data to identify key drivers
of dissatisfaction and recommend data-driven strategies to improve overall satisfaction.

## Objectives
- Identify factors influencing passenger satisfaction
- Analyze satisfaction by passenger segments
- Evaluate the impact of arrival delays on satisfaction
- Build a single-page interactive dashboard to support decision-making

## Dataset
- 120,000+ passenger records
- Key features include Satisfaction, Class, Type of Travel, Customer Type, Age, and Arrival Delay
- Data was cleaned and transformed using Power Query in Power BI

## Key Insights
- Business travelers report higher satisfaction than personal travelers
- Economy class passengers show the lowest satisfaction
- First-time customers are less satisfied than returning customers
- Satisfaction decreases significantly as arrival delays increase
- Younger passengers show lower satisfaction compared to older age groups

## Dashboard
The Power BI dashboard includes:
- KPIs: Satisfaction Rate, Dissatisfied Rate, Punctuality Rate, Satisfaction Gap, Total Passengers
- Visuals: Satisfaction by Class, Travel Type, Customer Loyalty, Age Category, Customer Type, and Arrival Delay Category
- Interactive slicers for dynamic filtering

## Recommendations
- Reduce moderate and severe arrival delays
- Improve Economy class passenger experience
- Enhance first-time passenger journey
- Monitor satisfaction metrics regularly using the dashboard

## Tools Used
- Power BI
- Power Query
